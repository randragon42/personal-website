Home File Server Ubuntu Server for Network File Share (2017 - 2018)

Network Attached Storage

With a hobby like photography, computer hard drives fill rather quickly. I needed a solution that would provide plenty of space for my current photo collection and the many photos to come. A home file server allows me to access my photos from anywhere and host my own personal cloud storage. I'm still learning about all the other possibilities I can use my server for.
