Home Automation: Raspberry Pi Hardware Hacking (2015)

Remote Controlled Outlets

A hardware hacking project to keep my busy over winter break. I purchased remote controlled outlets, gutted the controller, and hooked it up to the GPIO pins on a Raspberry Pi. A simple C program later and I was able to control the outlets from code.

Project Features

  • Control outlets from a simple webpage

Skills Learned

  • HTML, CSS, and JavaScript
  • Apache Web Server
