Lumos Computer Engineering Senior Design Project

Lumos: The Smart Light Switch

PCB design and assembly using TI Tiva Launchpad. All assembly done by hand using wave soldering. Finished product fit within a traditional light switch housing.

Project Features

  • Rotary Encoding Push Button with RGB LED
  • Microphone for clap-on/off
  • Reset Button
  • Ambient Light Sensor
  • TI CC3100 Wi-Fi Connectivity (Connects with Philips Hue and Android App)
  • IR Blaster (To control certain bulbs)
  • Relay - control conventional bulbs

Skills Learned

  • PCB design using Altium
  • Hardware Design
  • Circuit Design
  • Inter-Team Collaboration (Hardware and Software Teams)
